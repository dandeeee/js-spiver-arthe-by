Краски для помещений

Экстрамоющиеся краски

Моющиеся краски на водной основе широкого спектра применения защищают поверхность от образования плесени, грибков и прочих биологических повреждений.

SUPERBIANCO (Супербьянко)

Моющаяся водоэмульсионная краска для интерьеров, характеризуется осбой белизной. Материал очень легок в нанесении.

Нанесение: кисть, валик, краскопульт;
Разбавление: 40-50% водой;
Расход: 7-8 м.кв/л (в два слоя).

SUPERBIANCO EXTRA (СуперБьянко Экстра)

Краска водоэмульсионная для внутренних и наружных поверхностей.На основе сополимеров. Укрывистая, хорошо моется.

Нанесение: кисть, валик, краскопульт;
Разбавление: 40-50% водой;
Расход: 8-9 м.кв/л (в два слоя).

GENYUS STANDARD (Джениус стандарт)

Профессиональная супермоющаяся краска на водной основе с высокой укрывистостью для использования в интерьере и для фасадов. Особая матовость делает ее совершенно непрозрачной, что позволяет скрывать недостатки поверхности.

Нанесение: кисть, валик, краскопульт;
Разбавление: 40-50% водой;
Расход: 8-9 м.кв/л (в два слоя).

GENYUS SPECIAL (Специальный Джениус)

Супермоющаяся краска на водной основе в виде бархата для профессионального использования в интерьере и для фасадов. Имеет повышенную укрывистость, придает поверхности приятный бархатистый вид. Используется также как подложка под декоративные материалы.

Нанесение: кисть, валик, краскопульт;
Разбавление: 40-50% водой;
Расход: 9-10 м.кв/л (в два слоя).

MEGALIT

Акриловая краска с высокими показателями укрывистости. Экстрамоющаяся.

Нанесение: кисть, валик, краскопульт;
Разбавление: 10-20% водой;
Расход: 4-4,5 м.кв/л (в два слоя).

Дышащие краски

PITTURA EXTRA (Питтура Экстра)

Моющаяся водоэмульсионная краска для интерьеров с низким расходом и высокой производительностью, паропроницаемая.

Нанесение: кисть, валик, краскопульт;
Разбавление: 40% водой;
Расход: 6-7 м.кв/л (в два слоя).

COLOMBINA (Коломбина)

Тиксотропная паропроницаемая краска для интерьеров. Полумоющаяся, с хорошим расходом. Не оставляет подтеков, хорошо держится на кисти.

Нанесение: кисть, валик, краскопульт;
Разбавление: 40% водой;
Расход: 6-7 м.кв/л (в два слоя).

TRASPIRELLA (Траспирелла)

Паропроницаемая краска на водной основе. Идеальна для интерьеров. Легкая в нанесении. Не колеруется.

Нанесение: кисть, валик, краскопульт;
Разбавление: 30% водой;
Расход: 5-6 м.кв/л (в два слоя).

Против плесени

REOSAN SEMILUCIDO (РЕОСАН полуглянцевый)

Нанесение: кисть, валик, краскопульт;
Разбавление: 20-30% водой;

REOSAN OPACO (РЕОСАН матовый)

Разбавление: 20-30% водой;
Расход: 9-11 м.кв/л (в два слоя).

REOSAN OPACO TR (РЕОСАН матовый TR)

Нанесение: кисть, валик, краскопульт;
Разбавление: 20-30% водой;
Расход: 7-8 м.кв/л (в два слоя).

REOSAN ANTICONDENSA (РЕОСАН Антиконденсат)
Термоизолирующая краска-антиконденсат.

Содержит полые стеклянные микрошарики, которые обеспечивают эффективное термоизолирующее и тем самым, антиконденсатное действие.

Нанесение: кисть, валик, краскопульт;
Разбавление: 10-20% водой;
Расход: 4-4,5 м.кв/л (в два слоя).
