Грунты

Алкидные грунты

Позволяют нейтрализовать ржавчину и улучшить качество нанесения красок и эмалей.

CROMAR

Высококачественный грунт, предотвращающий ржавление металла.

Цвета: светло-серый, красный;
Нанесение: кисть, краскопульт;
Разбавление: 5-10% минеральным растворителем;
Расход: 8-10 м.кв/л (в два слоя).

CROMAR BIANCA

Высококачественный грунт, предотвращающий ржавление металла.

Нанесение: кисть, краскопульт;
Разбавление: 5-10% минеральным растворителем;
Расход: 8-10 м.кв/л (в два слоя).

CROMAR ZINC

Высококачественный грунт, содержащий фосфат цинка. Используется для нанесения на гальванизированные поверхности.

Цвета: белый, желтый;
Нанесение: кисть, краскопульт;
Разбавление: 10-20% минеральным растворителем;
Расход: 8-10 м.кв/л (в два слоя).

CROMARITE

Базовый матовый грунт.

Нанесение: кисть, краскопульт;
Разбавление: 5-10% минеральным растворителем;
Расход: 8-10 м.кв/л (в два слоя).

Грунты на водной основе

METALPRIMER

Высококачественная грунтовка на водной основе для железных и оцинкованных поверхностей. Особенно хорошо подходит для легких сплавов. Этот продукт имеет превосходную антикоррозионную в силу.

Нанесение: кисть, валик, краскопульт;
Разбавление: до 5% водой;
Расход: 10-12 м.кв/л на слой.

HIDROPAC

Грунтовка на водной основе для нанесения на цементные поверхности. Также отлично подходит для грунтования древесины и гипсокартонных плит.

Нанесение: кисть, валик, краскопульт;
Разбавление: до 5% водой;
Расход: 10-12 м.кв/л на слой.
