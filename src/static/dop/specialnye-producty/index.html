Специальные продукты

Капсулы из асбеста и цемента

REOSAN ANTIAMIANTO

Эластомерная краска на водной основе. Особенно подходит для связывания асбестовых или цементных поверхностей.

Нанесение: кисть, валик, краскопульт;
Разбавление:10-20% водой;
Расход: 0,25-0,3 м.кв/л на слой.

Специальные гидроизолирующие продукты

MULTIGUM

Жидкий акриловый защитный состав для террас. Доступен в белом, сером и красном цветах.

Нанесение: кисть, валик, краскопульт;
Разбавление:допускается 50% разбавление первого слояд; для второго слоя разбавление не требуется;
Расход: 2 м.кв/л на слой.

RIFLES

Синтетическая краска для алюминия. Также этот продукт имеет высокую отражательную способность и подходит для защиты битумных поверхностей.

Нанесение: кисть, валик, краскопульт;
Разбавление:не требуется;
Расход: 14 м.кв/л на слой.

Нетоксичные краски для дерева

TOP EPOXY

Многоцелевой эпоксидный лак. Подходит для полов промышленных и жилых помещений. Широко используется в качестве настенного покрытия в медицинской среде и пищевой промышленности.

Нанесение: кисть, валик, краскопульт;
Разбавление:5-10% водой;
Расход: 5-6,5 м.кв/л на слой.

Барьерные материалы против вздутия

TECNOSILAN

Полисилоксановый химический барьерьерный состав. Этот двухкомпонентный продукт используется для инъекций против вздутий поверхностей.
